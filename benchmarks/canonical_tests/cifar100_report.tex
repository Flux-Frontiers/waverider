\section{Key Findings}

\begin{itemize}[leftmargin=*,itemsep=3pt]
  \item \textbf{Best architecture:} ManifoldResNet-UB (w*=118) \textemdash\ test accuracy $0.3829 \pm 0.0377$
  \item \textbf{vs Standard:} $+0.3308$ (33.08\,pp) accuracy gain
  \item \textbf{Parameter reduction:} 5.8$\times$ fewer parameters (644,262 vs 3,722,852)
  \item \textbf{Parameter efficiency:} 42.5$\times$ improvement (0.0006 vs 0.0000 acc/Kparam)
  \item \textbf{Manifold compression:} 3,072D $\rightarrow$ 19D (99.4\%\ of ambient dimensions carry no structure)
\end{itemize}
